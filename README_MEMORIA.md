
Juego de Memoria - agregado por ChatGPT
Archivo: memoria.py

Descripción:
- Juego de memoria por consola (terminal).
- Implementa recursividad: la función play_round se llama a sí misma hasta que el juego termina.
- Usa diccionarios para mapear posiciones (p.ej. 'A1') a valores y para seguimiento de casillas reveladas.
- Validaciones complejas: formato de tamaño, formato de posiciones, posiciones fuera de rango,
  manejo de casillas ya reveladas, uso de pistas y límites de tamaño.

Uso:
1) Extraer el ZIP.
2) Ejecutar: python3 memoria.py
3) Seguir instrucciones en pantalla.

Nota: El script está pensado para ejecutarse en una terminal que soporte emojis.
